# ECommerce

 Este projeto foi gerado utilizando o Angular CLI, versão 22.0.4.

## Servidor de desenvolvimento

Para iniciar um servidor de desenvolvimento local, execute:

```bash
ng serve
```

Quando o servidor estiver em execução, abra o navegador  e acesse `http://localhost:4200/`.A aplicação sera recarregada automaticamente sempre que voce modificar qualquer um dos arquivo de codigo-fonte.

## Estrutura inicial de código

:A Angular CLI inclui poderosas ferramentas de geração de código (code scaffolding). Para gerar um novo componente, execute:

```bash
ng generate component component-name
```

para ver a lista completa de garedores (schematics disponíves, como  `components`, `directives`, ou `pipes`), execute:

```bash
ng generate --help
```

## Compilação 

para gerar a versão de compilação do projeto,execute

```bash
ng build
```

isso compilara o seu projeto e armazenará os arquivos gerados da compilação no diretório  `dist/` por pardão, a compilação para produção otimiza a aplicação para oferecer melhor desempenho e velocidade.

## Executando testes unitários 

para executar os testes unitários como o executor de testes[Vitest](https://vitest.dev/) utilize o siguente comando:

```bash
ng test
```

## Executando testes de ponta a ponta

para executar os testes de ponta a ponta(end-to-end ou e2e), execute:

```bash
ng e2e
```

A Angular CLI não inclui um framework de testes de ponta a ponta (end-to-end ou e2e) por padrão. Você pode escolher o que melhor atender às suas necessidades.
